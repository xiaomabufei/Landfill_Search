# 填埋场信息搜索项目 — 总览

## 项目目标

通过自动化网络搜索，逐国获取全球填埋场的详细运营信息和环境数据，构建一个可溯源、可验证的填埋场数据库。

## 快速体验

```bash
# 方式 A: macOS App（普通用户）
./build_app.sh                    # 打包
open "dist/Landfill Search.app"   # 双击运行

# 方式 B: Web UI（浏览器）
python web_app.py                 # 访问 http://localhost:5000

# 方式 C: 源码运行（开发者）
./run_test.sh                     # 一键测试
python gui_app.py                 # GUI 模式
python main.py data/raw/ITA.xlsx --scrape --headed  # 命令行模式
```

> **平台**: 目前仅在 macOS 上测试通过。

详见 [README.md](../README.md)。

## 数据指标

### 核心指标（7 项）

| # | 指标 | 字段名 | 取值范围 |

## 工作流程

1. **读取原始数据** — 自动识别文件格式（xlsx/csv/tsv/json），智能映射列名
2. **自动搜索** — Playwright 驱动 Chromium，批次爬取 Google/Scholar/Maps
3. **信息提取** — 英语+意大利语双语关键词规则匹配，自动推断来源类型
4. **数据检查** — 格式校验、逻辑校验（如 start_year > final_year）、完整性检查
5. **人工复核** — 查看日志中 🔴 标记项，错误日志独立保存便于排查
6. **生成输出** — JSON（可溯源）+ HTML（地图+图表+表格）+ Markdown 日志

## 技术栈

| 用途 | 工具 | 状态 |
|------|------|------|
| 自动搜索 | Playwright + Chromium/Chrome/WebKit | ✅ |
| Google Search | google_search.py（适配 2026 DOM） | ✅ |
| Google Scholar | google_scholar.py | ✅ |
| Google Maps | google_maps.py | ✅ |
| 信息提取 | extractor.py（英语+意大利语规则） | ✅ |
| 浏览器检测 | browser_detect.py（自动扫描本机浏览器） | ✅ |
| 多进程并行 | multiprocessing + worker.py | ✅ |
| 数据读取 | reader.py（xlsx/csv/tsv/json） | ✅ |
| 数据检查 | validator.py（4 维检查 + 报告生成） | ✅ |
| 可视化 | Leaflet.js + CartoDB + Chart.js | ✅ |
| 日志 | PipelineLogger（时间戳+进度+ETA+折叠+错误日志） | ✅ |
| 一键启动 | run_test.sh（自动环境检测+依赖安装） | ✅ |

| Web UI | web_app.py（Flask，浏览器访问 localhost:5000） | ✅ |
| App 打包 | build_app.sh + PyInstaller | ✅ |
