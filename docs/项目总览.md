# 填埋场信息搜索项目 — 总览

## 项目目标

自动化搜索**全球任意国家**的填埋场详细运营信息和环境数据，构建可溯源、可验证的填埋场数据库。只需提供一个填埋场列表文件，系统自动完成搜索→提取→校验→可视化全流程。

> 当前以意大利（179 个填埋场）为首个测试国家，系统架构已支持任意国家扩展。

## 快速体验

```bash
# 方式 A: macOS App
./build_app.sh && open "dist/Landfill Search.app"

# 方式 B: Web UI
python web_app.py        # 访问 http://localhost:5000

# 方式 C: 命令行
./run_test.sh            # 一键运行
python main.py data/raw/YOUR_DATA.xlsx --scrape --headed
```

> **平台**: 目前仅在 macOS 上测试通过。

详见 [README.md](../README.md)。

## 工作流程

1. **读取原始数据** — 自动识别文件格式（xlsx/csv/tsv/json），智能映射列名
2. **自动搜索** — Playwright 驱动多浏览器，批次爬取 Google/Scholar/Maps
3. **信息提取** — 多语言关键词规则匹配（目前支持英语+意大利语，可按国家扩展）
4. **数据检查** — 格式校验、逻辑校验、完整性检查（自动捕获如 start_year > final_year）
5. **人工复核** — 查看日志中 🔴 标记项
6. **生成输出** — JSON（可溯源）+ HTML（地图+图表+表格）+ 日志

## 技术栈

| 用途 | 工具 | 状态 |
|------|------|------|
| 自动搜索 | Playwright + Chromium/Chrome/WebKit | ✅ |
| Google Search/Scholar/Maps | 各独立爬虫模块 | ✅ |
| 信息提取 | extractor.py（多语言关键词规则） | ✅ |
| 浏览器检测 | browser_detect.py（自动扫描本机浏览器） | ✅ |
| 多进程并行 | multiprocessing + worker.py | ✅ |
| 数据读取 | reader.py（5 种格式 + 智能列名映射） | ✅ |
| 数据检查 | validator.py（4 维检查 + 报告生成） | ✅ |
| 可视化 | Leaflet.js + CartoDB + Chart.js | ✅ |
| 日志 | PipelineLogger（时间戳+进度+ETA+折叠+错误日志） | ✅ |
| 使用方式 | macOS App / Web UI / CLI | ✅ |
