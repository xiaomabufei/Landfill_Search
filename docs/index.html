Features
End-to-end automated pipeline from raw data to interactive visualization
🔍

Automated Search

Playwright drives Chrome/WebKit to scrape Google Search, Google Scholar, and Google Maps. Bilingual keyword extraction (EN + IT).

✅

Data Validation

4-dimension check: format, logic (start_year ≤ final_year), completeness, and source verification. Errors highlighted in bold red.

🗺️

Map Visualization

CartoDB + Leaflet.js interactive map. Color-coded markers, click for details with source links. Chart.js statistics.

🔄

Resume & Incremental

Checkpoint resume — interrupted searches continue from where they stopped. Results saved after each landfill.

🖥️

Multi-Browser Parallel

Auto-detects installed browsers (Chrome, WebKit, Firefox). Parallel workers with staggered delays to avoid detection.

📋

Rich Logging

Timestamped logs with progress bar, ETA, collapsible details. Separate error log for quick triage.

Architecture
Simple pipeline, powerful automation
