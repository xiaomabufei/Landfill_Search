# 4. 总结模块约束

## 目标

对验证后的数据生成 **JSON 详细数据** 和 **HTML 可视化展示**。

---

## 输出格式

| 格式 | 路径 | 用途 |
|------|------|------|
| JSON | `output/{CODE}.json` | 结构化数据，每项附来源链接 |
| HTML | `output/html/{CODE}.html` | 地图 + 图表 + 表格交互展示 |

## HTML 可视化功能

| 功能 | 技术 | 状态 |
|------|------|------|
| 地图标注 | Leaflet.js + CartoDB Light | ✅ |
| 类型着色 | 蓝=sanitary，红=dump，灰=未知 | ✅ |
| 弹窗详情 | 点击标注点显示指标 + ref 链接 | ✅ |
| 统计卡片 | 总数、Gas 收集数、类型分布、完整率 | ✅ |
| 类型饼图 | Chart.js 环形图 | ✅ |
| 技术柱状图 | Chart.js 柱状图 | ✅ |
| 数据表格 | 全指标 + Badge + 缺失标记 | ✅ |
| 独立运行 | 纯静态 HTML，CDN 加载 | ✅ |

## 运行方式

```bash
# 通过 Pipeline 自动生成
python main.py data/raw/ITA.xlsx

# 单独生成
python src/summary/html_generator.py
```
