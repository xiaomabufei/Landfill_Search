# 2. 搜索模块约束

## 目标

通过 Playwright 驱动本机浏览器，自动爬取 Google Search / Scholar / Maps，获取**任意国家**填埋场的详细指标信息。支持多浏览器并行、断点续搜、增量保存。

---

## 搜索引擎

| 引擎 | 代码文件 | 用途 | 状态 |
|------|---------|------|------|
| Google Search | `google_search.py` | 政府报告、新闻、官方网站 | ✅ |
| Google Scholar | `google_scholar.py` | 学术文献、技术参数 | ✅ |
| Google Maps | `google_maps.py` | 位置确认、运营状态 | ✅ |
| 国家级数据库 | 按国家扩展 | 官方登记数据（如意大利 ISPRA） | 🔲 |
| Google API | 备用 | 免费 100 次/天 | 🔲 |

## 搜索命令

```bash
# 基础搜索（有头模式，首次需手动过验证码）
python -m src.search.scrape_runner data/raw/YOUR_DATA.xlsx --headed

# 多浏览器并行
python -m src.search.scrape_runner data/raw/YOUR_DATA.xlsx --headed --workers 3

# 搜索指定范围（断点续搜）
python -m src.search.scrape_runner data/raw/YOUR_DATA.xlsx --headed --start 0 --end 50

# 只用 Google Search
python -m src.search.scrape_runner data/raw/YOUR_DATA.xlsx --headed --engines google

# 全流程：搜索 + 检查 + 展示
python main.py data/raw/YOUR_DATA.xlsx --scrape --headed
```

### 参数说明

| 参数 | 默认 | 说明 |
|------|------|------|
| `--headed` | 无头 | 显示浏览器窗口（首次必须用） |
| `--workers N` | 1 | 并行 Worker 数 |
| `--start N` | 0 | 起始索引 |
| `--end N` | 全部 | 结束索引 |
| `--engines` | google,scholar,maps | 启用的搜索引擎 |
| `--batch-size` | 10 | 批次大小 |

## 信息提取规则

从搜索结果中通过**多语言关键词匹配**提取结构化数据。目前支持英语 + 意大利语，新国家需扩展对应语言关键词。

| 提取项 | 英语关键词（通用） | 意大利语关键词（示例） |
|--------|------------------|----------------------|
| landfill_type | sanitary landfill, dump | discarica, rifiuti non pericolosi |
| has_gas_collection | biogas, gas collection | captazione, recupero energetico |
| gas_technology | flaring, electrification | torcia, cogenerazione |
| start_year | opened, established, since | anno apertura, attiva dal |
| final_year | closed, closure | chiusa, chiusura, esaurita |

### 扩展新语言

在 `src/search/extractor.py` 的各提取函数中添加对应语言的关键词即可，无需修改其他代码。

来源类型自动推断：
- `government_report`：.gov, regione., comune., gobierno
- `academic_paper`：scholar, sciencedirect, springer
- `news`：news, today, notizie, press
- `database`：wiki, catasto, database

## 反爬策略

| 策略 | 说明 |
|------|------|
| 随机延迟 | 请求间 2-6s，批次间 30-60s |
| 浏览器状态保存 | cookies 持久化到 `output/browser_state/` |
| 多 User-Agent | Chrome/Firefox/Safari 各用对应 UA |
| 验证码检测 | 自动识别 reCAPTCHA，暂停等待手动通过 |
| 多引擎分散 | 不同浏览器引擎不同指纹 |
