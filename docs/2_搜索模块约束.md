# 2. 搜索模块约束

## 目标

通过 Playwright 驱动本机浏览器，自动爬取 Google Search / Scholar / Maps，获取每个填埋场的详细指标信息。支持多浏览器并行、断点续搜、增量保存。

---

## 搜索引擎

| 引擎 | 代码文件 | 用途 | 状态 |
|------|---------|------|------|
| Google Search | `google_search.py` | 政府报告、新闻、官方网站 | ✅ |
| Google Scholar | `google_scholar.py` | 学术文献、技术参数 | ✅ |
| Google Maps | `google_maps.py` | 位置确认、运营状态 | ✅ |
| Google API | 备用 | 免费 100 次/天 | 🔲 |

## 搜索命令

```bash
# 基础搜索（有头模式，首次需手动过验证码）
python -m src.search.scrape_runner data/raw/ITA.xlsx --headed

# 多浏览器并行（自动检测 Chrome/WebKit 等）
python -m src.search.scrape_runner data/raw/ITA.xlsx --headed --workers 3

# 搜索指定范围（断点续搜，已搜过的自动跳过）
python -m src.search.scrape_runner data/raw/ITA.xlsx --headed --start 0 --end 50

# 只用 Google Search（更快，不搜 Scholar/Maps）
python -m src.search.scrape_runner data/raw/ITA.xlsx --headed --engines google

# 通过 main.py 一键搜索+检查+展示
python main.py data/raw/ITA.xlsx --scrape --headed
```

### 参数说明

| 参数 | 默认 | 说明 |
|------|------|------|
| `--headed` | 无头 | 显示浏览器窗口（首次必须用，过验证码） |
| `--workers N` | 1 | 并行 Worker 数 |
| `--start N` | 0 | 起始索引 |
| `--end N` | 全部 | 结束索引 |
| `--engines` | google,scholar,maps | 启用的搜索引擎 |
| `--batch-size` | 10 | 批次大小（批次间休息 30-60s） |

## 信息提取规则

从搜索结果中通过**英语+意大利语双语关键词匹配**提取结构化数据：

| 提取项 | 英语关键词 | 意大利语关键词 |
|--------|-----------|--------------|
| landfill_type | sanitary landfill, dump | discarica, rifiuti non pericolosi |
| has_gas_collection | biogas, gas collection | captazione, recupero energetico, biometano |
| gas_technology | flaring, electrification | torcia, energia elettrica, cogenerazione |
| start_year | opened, established, since | anno apertura, attiva dal |
| final_year | closed, closure | chiusa, chiusura, esaurita |

来源类型自动推断：
- `government_report`：.gov, regione., comune.
- `academic_paper`：scholar, sciencedirect, springer
- `news`：news, today, notizie
- `database`：wiki, catasto, ispra

## 反爬策略

| 策略 | 说明 |
|------|------|
| 随机延迟 | 请求间 2-6s，批次间 30-60s |
| 浏览器状态保存 | cookies 持久化到 `data/browser_state/` |
| 多 User-Agent | Chrome/Firefox/Safari 各用对应 UA |
| 验证码检测 | 自动识别 reCAPTCHA，暂停等待手动通过 |
| 多引擎分散 | 不同浏览器引擎不同指纹 |
| Worker 错开启动 | 间隔 3s，避免同时触发 |

---

## 多浏览器支持

`browser_detect.py` 自动扫描本机浏览器：

| 检测路径 | 引擎 | Channel |
|---------|------|---------|
| /Applications/Google Chrome.app | chromium | chrome |
| /Applications/Microsoft Edge.app | chromium | msedge |
| /Applications/Firefox.app | firefox | — |
| /Applications/Brave Browser.app | chromium | chrome |
| Playwright 内置 Chromium | chromium | — |
| Playwright 内置 WebKit | webkit | — |

并行搜索时优先选择不同引擎类型，减少指纹重复。

| `google_scholar.py` | Google Scholar 爬虫 |

| `reader.py` | 多格式读取（xlsx/csv/tsv/json） |
| `query_builder.py` | 搜索关键词模板 |
| `output_writer.py` | JSON 输出 |
